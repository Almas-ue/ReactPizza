# React-Pizza
Пет-проект по react-typescript, показать скиллы.

### Используемые технологии и библиотеки: 
- React
- Typescript
- Scss
- normalize.css
- Snadcn
- Tailwind
- Vite
- Lucide React
- React-Hook-Form

### Для запуска проекта
---
```
npm start
```

Open http://localhost:3000 with your browser to see the result.
